== Description ==

This Plugin is use for to change WP Admin login page Icon, Title and URL Link. Once you have activated plugin one menu item Add Image will be activated after clicking on Add Image Upload Image panel will be open now you can upload image through that panel if image is already exist it update the previous image link otherwise it insert new image.
